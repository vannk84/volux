Pro · 标签页覆盖

同一站点。三个标签。三种音量。

多数扩展只允许一个网站一个音量。Volux Pro 让任何站点 — YouTube、TikTok、Facebook、Spotify、Twitch — 的每个标签页独立保持音量,同时不影响站点默认值。

  • 展开 点击任一域名上的 N TABS 徽章,展开每个打开的标签页及其独立滑块。
  • 覆盖 拖动某个标签页的滑块即可赋予独立音量。未自定义的标签页仍沿用站点默认。
  • 清除 关闭标签页或切换到其他站点 — 覆盖自动清除。站点默认保持整洁。

06 · 问答

常见问题。

点击任一问题展开。

为什么每次刷新后浏览器音量都会重置?
浏览器默认不保存每个网站的音量。每次刷新或重开标签页,音量会回到 100%。Volux 通过自动记住你为每个网站设定的音量来解决这个问题,你再也不用反复调整。
如何为 YouTube 和 Spotify 设置不同音量?
每个网站都有独立的音量滑块。把 YouTube 设为 30%、Spotify 设为 80% 或任何你喜欢的值。Volux 会自动保存并在每次访问时应用。
为什么 YouTube 总是比其他网站大声?
不同网站音频的母带电平不同。YouTube 视频通常比其他内容更响。Volux 让你把 YouTube(或任何吵闹站点)永久固定在较低音量,不再被震到。
关闭浏览器后 Volux 还记得我的设置吗?
是的。设置保存在浏览器本地存储中。关闭 Chrome、重启电脑、几天后回来 — Volux 仍会按你设置的音量应用。
我能控制 Zoom、Google Meet、Microsoft Teams 的音量吗?
可以。Volux 适用于任何播放音频的网站,包括视频会议应用。让会议音量保持舒适,同时其他标签页播放不同音量。
Volux 安全吗?需要哪些权限?
Volux 是开源且注重隐私的。只请求在网页上控制音频所需的权限。不收集任何个人数据、浏览记录或分析。代码可在 GitHub 上审阅。
Free 和 Pro 有什么区别?
Free 版支持控制 2 个网站 — 足够试用。Pro 版($2 一次性)解锁无限网站、自动识别媒体站点、扫描所有打开标签页。终身购买,无订阅。
Volux 能支持同一网站的多个标签页吗?
能 — 两种模式。默认下,同一站点的所有标签页共享一个音量。Pro 版下,你可以覆盖任一标签页:同一站点三个标签页可以同时保持三种不同音量,未自定义的仍沿用站点默认。
我能为同一站点(YouTube、TikTok、Facebook…)的多个标签页设置不同音量吗?
可以,使用 Volux Pro。适用于任何有音频的站点 — YouTube、TikTok、Facebook、Spotify、Twitch、SoundCloud、新闻站点,到处可用。点击域名卡片上的“N tabs”徽章展开按标签列表,然后拖动任一标签页的滑块赋予独立音量。关闭标签页或切换到其他站点时,覆盖自动清除,站点默认值保持整洁。
