ブラウザの音、ついに言うことを聞く

一度だけ設定。ずっと記憶。

Volux は サイトごとの音量を記憶します — YouTube 30%、Facebook 10%、Zoom 100%。Pro 版ではどのサイトの各 タブ も個別の音量を保持できます。

01 · 機能一覧

音の悩みを解決します。

すべてのサイトを望みの音量に。毎回。一度設定すれば、摩擦ゼロ。

/ 01

音量の自動保存

YouTube を 30%、Facebook を 10% に — Volux が永続的に記憶。拡張機能を開き直す必要なし。

/ 02

サイトごとに異なる音量

Zoom 会議、YouTube チュートリアル、バックの Spotify — それぞれベストな音量で、干渉なし。

/ 03

音量リセットからの解放

ページを再読込。ブラウザを閉じる。来週また起動。あなたの音量はそのまま。

/ 04

うるさいサイトを制御

どうしても音が大きいサイト。一度低く固定すれば、永遠に解決。

/ 05

セッションをまたいで動作

音量はブラウザにローカル保存。アカウント不要。クラウド往復なし。

/ 06

軽量

完全にクライアントサイドで動作。キロバイト単位。バックグラウンドのポーリング・テレメトリなし。

Pro · タブ別オーバーライド

同じサイト。3 つのタブ。3 つの音量。

ほとんどの拡張機能はサイトごとに 1 つの音量しか許しません。Volux Pro なら、どのサイト — YouTube、TikTok、Facebook、Spotify、Twitch — の各タブも、サイト既定値に触れずに独立した音量を保持できます。

  • 展開 任意のドメインの N TABS バッジをタップして、開いている各タブとその専用スライダーを表示。
  • 上書き 任意のタブのスライダーを動かして独自の音量を設定。未カスタマイズのタブには引き続きサイト既定値が適用されます。
  • クリア タブを閉じるか別サイトへ移動すると、上書きは自動でクリア。サイト既定値はクリーンに保たれます。

03 · セットアップ

使い方。

小さな 3 ステップ。その後は自動。ずっと。

1

拡張機能をインストール

ワンクリックで Chrome か Firefox に追加。アカウント不要。設定不要。ツールバーにアイコンが即表示。

2

任意のサイトで調整

Volux アイコンをクリックしてスライダーをドラッグ。YouTube 30%、ポッドキャスト 80%。即時反映。

3

Volux が記憶

完了。音量はローカル保存され、毎回自動で再適用 — 再読込、再起動、数週間後でも。

Pro のヒント:既定では、同一サイトのすべてのタブは 1 つの音量を共有 — YouTube、TikTok、Facebook、Spotify、どこでも同じ。Pro ではタブ数バッジを展開して各タブに独自の音量を設定可能。サイト既定値は未カスタマイズのタブに引き続き適用されます。

06 · 質問

よくある質問。

質問をタップすると開きます。

再読込のたびに音量が元に戻るのはなぜ?
ブラウザは既定でサイトごとの音量を保存しません。再読込やタブの再オープンで 100% に戻ります。Volux はサイトごとの希望音量を自動で記憶し、再調整の必要をなくします。
YouTube と Spotify で別々の音量を設定するには?
各サイトが独立したスライダーを持ちます。YouTube 30%、Spotify 80%、好みの値を設定してください。Volux が自動保存し、次回訪問時に適用します。
なぜ YouTube は他サイトよりうるさいの?
サイトごとに音声マスタリングのレベルが異なります。YouTube はしばしば大きめ。Volux なら YouTube (や任意の大音量サイト) を永続的に低音量で固定可能。もう驚かされません。
ブラウザを閉じても設定を記憶しますか?
はい。設定はブラウザのローカルストレージに保存されます。Chrome を閉じ、PC を再起動し、数日後に戻っても、Volux は設定どおりの音量を適用します。
Zoom、Google Meet、Microsoft Teams の音量も操作できますか?
はい。Volux は音声を再生するどんなサイトでも動作します (ビデオ会議アプリを含む)。会議音声を快適な音量に保ちつつ、他タブは別音量で再生できます。
Volux は安全?どんな権限が必要?
Volux はオープンソースでプライバシーに配慮。ページ上の音声制御に必要な権限のみ要求します。個人データ、閲覧履歴、解析は一切収集しません。コードは GitHub で確認できます。
Free と Pro の違いは?
Free は 2 サイトを制御 — お試しに最適。Pro ($2 買い切り) は無制限サイト、メディアサイトの自動検出、開いているタブのスキャンを解除。生涯購入、サブスクなし。
同じサイトの複数のタブで動作しますか?
はい — 2 つのモードがあります。既定では、同一サイトのすべてのタブが 1 つの音量を共有。Pro では個別タブを上書き可能。同じサイトの 3 タブが同時に 3 つの異なる音量を保持でき、サイト既定値は未カスタマイズのタブに引き続き適用されます。
同じサイト (YouTube、TikTok、Facebook…) の複数タブに異なる音量を設定できますか?
はい、Volux Pro で可能。音声のある任意のサイトで動作 — YouTube、TikTok、Facebook、Spotify、Twitch、SoundCloud、ニュースサイトなど。ドメインカードの「N tabs」バッジをクリックしてタブ別リストを展開し、任意のタブのスライダーをドラッグして独自音量を付与。タブを閉じるか別サイトに移動すると上書きはクリアされ、サイト既定値はクリーンに保たれます。
