## Action Required: Demo Video

### What to Record
Create a 1-3 minute screen recording showing:

1. **Installation** (15 sec)
   - Go to Chrome Web Store or Firefox Add-ons
   - Click "Add to Chrome/Firefox"
   - Show the extension icon appearing

2. **Basic Usage** (30 sec)
   - Open the popup
   - Add a domain (e.g., youtube.com)
   - Adjust the volume slider
   - Show volume changing on the website
   - Use the mute button

3. **Pro Features** (30 sec)
   - Show the domain limit for free users
   - Click upgrade button
   - Show the license activation flow
   - Demonstrate unlimited domains after activation
   - Show the "Scan Tabs" feature

4. **Settings Page** (15 sec)
   - Open the options/settings page
   - Show license status
   - Show usage statistics

### Recommended Recording Tools
- **Loom** (free, easy sharing): https://loom.com
- **OBS Studio** (free, local): https://obsproject.com
- **macOS**: Cmd+Shift+5 (built-in)
- **Windows**: Win+G (Xbox Game Bar)

### Video Tips
- Use a clean browser profile (no personal bookmarks)
- Close unnecessary tabs
- Use 1080p resolution if possible
- Add soft background music (optional)
- Keep it under 3 minutes

### Where to Host the Video
- **YouTube** (unlisted link)
- **Loom** (direct link)
- **Google Drive** (shareable link)
- **Vimeo** (basic account is free)
